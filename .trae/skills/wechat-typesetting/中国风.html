欢迎使用微信 Markdown 编辑器

沉浸创作，美由心生。我们致力于为您提供一个简洁、高效、美观的公众号文章排版工具。

您只管专注于文字，排版的美感，我们来呈现。

✨ 主要功能

  • • 实时预览：左侧编辑，右侧即时查看排版效果。
  • • 一键复制：轻松复制富文本格式，直接粘贴到公众号后台。
  • • 精美样式：内置精心调校的微信主题，确保最佳阅读体验。
  • • 代码高亮：支持多种编程语言的语法高亮。
  • • Markdown 兼容：全面支持标准及扩展 Markdown 语法。
  • • API 转换：提供强大的 RESTful API，支持批量转换和第三方集成。
  • • MCP 协议：支持 Model Context Protocol，可与 AI 助手无缝协作创作。

🚀 如何使用

🖊️ 基础编辑

  1. 1. 在左侧的编辑区输入或粘贴您的 Markdown 内容。
  2. 2. 在右侧的预览区查看实时渲染的效果。
  3. 3. 点击右上角的"复制"按钮，然后到微信公众号后台粘贴即可。

🔗 API 调用

通过 RESTful API 实现自动化转换和批量处理：

# 转换 Markdown 为微信格式
curl -X POST /api/convert \
  -H "Content-Type: application/json" \
  -d '{"markdown": "# 标题", "theme": "default"}'

🤖 MCP 协作

开源项目地址：md2wechat-mcp-server

支持与 Claude、Cursor 等 AI 工具协作创作：

  1. 1. 配置 MCP：配置 MCP Server 地址。
  2. 2. 智能生成：AI 工具可直接调用编辑器进行内容创作。
  3. 3. 实时协作：在对话中实时预览和调整排版效果。
