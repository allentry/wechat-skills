使用前设置（必看必配置）：
1、把两个mcp配置好，Chrome DevTools MCP（mcp市场可以直接找到，添加即可），lark-mcp（这个mcp市场没有，需要自行搜索教程，关键词：飞书 mcp。不安装也没什么影响）
2、gemini-image，这个skill首先需要到下面网址充值，并生成apikey，填写到文档：\.trae\skills\gemini-image\config\secrets.md。
这里使用的是banana pro，4K，2毛钱一张图。
到这个网站充值，获取apikey：https://ismaque.org/register?aff=Ikkn
3、wechat-typesetting，这个skill首先需要到下面的微信开发者平台后台获取APPID、APPSECRET
填写到.trae\skills\wechat-typesetting\config\secrets.md。
然后随便找个查IP地址的网站（比如https://ip138.com/），把本机IP地址，填写到微信开发者平台后台的API IP白名单里面。
微信开发者平台后台：https://developers.weixin.qq.com/
